# Aesthetic Playbook

Pick ONE direction and commit. Swap colors/fonts if needed, but keep the intent. Choose a single **signature move** that someone will remember.

## Font Rule (avoid default vibes)
- If the repo already has a font system, stay consistent unless asked to change it.
- Otherwise: pick a characterful display font + a legible body font. Avoid overused “default tech” pairings.

## Brutalist Raw (high-contrast, rule-heavy)
- Fonts: Display `Bricolage Grotesque` / `Fraunces`; Body `Atkinson Hyperlegible` / `IBM Plex Sans`; Mono `IBM Plex Mono`.
- Colors: Ink/ivory with one acidic accent (e.g. tomato, chartreuse).
- Signature moves: oversized numeric index badges; thick 2px rules; hard invert hover.
- Backgrounds: checkerboard corner, dotted matrix, a single diagonal stripe.

## Editorial / Magazine (luxury, typographic hierarchy)
- Fonts: Display `Instrument Serif` / `Cormorant Garamond`; Body `Source Serif 4` / `Newsreader`; Mono `DM Mono`.
- Colors: warm off-white, deep ink, one burnt accent (rust/saffron).
- Signature moves: pull-quote rail; hairline rules; “issue number” metadata strip.
- Backgrounds: subtle paper grain, soft vignette.

## Techno-Gothic (dark, minimal, sharp accents)
- Fonts: Display `Unbounded` / `Syne`; Body `Work Sans` / `Barlow`; Mono `JetBrains Mono`.
- Colors: near-black base with cold accent (cyan/acid green) + muted steel.
- Signature moves: corner ticks, scanline sheen on hover, “system status” chips.
- Backgrounds: diagonal hatch, faint grid, tiny starfield.

## Neo-Poster (bold blocks, loud hierarchy)
- Fonts: Display `Bebas Neue` / `Archivo Black`; Body `Noto Sans`; Mono `Space Mono` (only if it fits the concept).
- Colors: high-sat primaries on warm neutral, with black rules.
- Signature moves: torn-paper edge (CSS mask vibe); huge headline with tight tracking; sticker CTA.
- Backgrounds: halftone dot field, off-register shadows.

## Soft Craft (warm, tactile, friendly)
- Fonts: Display `Sora` / `Epilogue`; Body `Alegreya Sans` / `Manrope`; Mono optional.
- Colors: warm cream, terracotta accent, sage secondary.
- Signature moves: stitched borders; stacked cards; gentle floating blobs (reduced-motion safe).
- Backgrounds: irregular gradient blobs + low-opacity noise.

## Industrial / Utilitarian (labeled panels, measured density)
- Fonts: Display `Archivo` / `Anton`; Body `Barlow` / `IBM Plex Sans`; Mono `JetBrains Mono`.
- Colors: cold neutrals with hazard accent (amber) and a secondary teal.
- Signature moves: panel labels; coordinate tags; “spec table” sectioning.
- Backgrounds: repeating hatch, soft inner glows for depth.

## How to Choose (fast)
1) Choose the emotional goal (luxe, raw, calm, severe, playful).
2) Pick type first (display sets the tone).
3) Pick one signature move and apply it in 2–3 places max.
4) Keep everything else quiet and supportive.
